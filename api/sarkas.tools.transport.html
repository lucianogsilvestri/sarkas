static diffusion(params, phase='production', compute_acf=True, no_slices=1, plot=True, show=False, figname=None, **kwargs)[source]¶

Calculate the self-diffusion coefficient from the velocity auto-correlation function.

Parameters

  • params (sarkas.core.Parameters) – Simulation’s parameters.

  • phase (str, optional) – Phase to analyze. Default = ‘production’.

  • compute_acf (bool, optional) – Flag for recalculating the ACF. Default = True. If False it will read in the data from the dataframe.

  • no_slices (int, optional) – Number of slices of the simulation. Default = 1.

  • plot (bool, optional) – Flag to plot transport coefficient with corresponding autocorrelation function. Default = True.

  • show (bool, optional) – Flag for prompting plot to screen.

  • figname (str, optional) – Name with which to save the file. It automatically saves it in the correct directory.

  • **kwargs – Arguments to pass sarkas.tools.observables.VelocityAutoCorrelationFunction()

Returns

coefficient – Pandas dataframe containing the value of the transport coefficient as a function of integration time.

Return type

pandas.DataFrame
